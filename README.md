# Bitcoin Market Sentiment vs Trader Performance Analysis

## Project Overview
This project analyzes the relationship between Bitcoin market sentiment and trader performance using historical Hyperliquid trading data and the Bitcoin Fear & Greed Index.

## Datasets
- Historical Trader Data
- Bitcoin Fear & Greed Index

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- SciPy
- Jupyter Notebook

## Project Workflow
1. Data Cleaning
2. Data Merging
3. Exploratory Data Analysis
4. Trader Performance Analysis
5. Market Sentiment Analysis
6. Statistical Testing
7. Visualizations
8. Business Insights
9. Recommendations
